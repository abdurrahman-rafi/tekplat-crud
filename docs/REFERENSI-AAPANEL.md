# Referensi aaPanel yang Dipakai

Panduan deployment dalam repo ini disusun dengan asumsi berdasarkan:

1. PDF lokal tugas: [PPT CRUD Database di aaPanel (1).pdf](../PPT%20CRUD%20Database%20di%20aaPanel%20%281%29.pdf)
2. Dokumentasi aaPanel tentang Proxy Project:
https://www.aapanel.com/docs/Function/proxy.html
3. Dokumentasi aaPanel tentang Go Project:
https://www.aapanel.com/docs/Function/go.html
4. Contoh deploy resmi aaPanel untuk Go Project:
https://www.aapanel.com/docs/Function/Tutorial/go-project-deploy-gogs.html

## Batasan

- Repo ini dibuat tanpa akses langsung ke Ubuntu VM atau instance aaPanel nyata.
- Karena itu, langkah deployment ditulis berbasis dokumentasi resmi aaPanel dan asumsi alur yang paling masuk akal untuk praktikum.
- Jika tampilan menu aaPanel pada mesin mahasiswa sedikit berbeda, ikuti istilah menu yang paling dekat, terutama `Website`, `Go Project`, `Domain Manager`, `Mapping`, dan `SSL`.
