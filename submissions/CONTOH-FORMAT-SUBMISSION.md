# Format Submission

## Yang wajib dikumpulkan

1. Link repository hasil pengerjaan.
2. Screenshot halaman web yang sudah bisa diakses dari browser melalui domain/IP VM.
3. Screenshot bukti operasi Create.
4. Screenshot bukti operasi Read.
5. Screenshot bukti operasi Update.
6. Screenshot bukti operasi Delete.
7. File SQL dump hasil akhir database, misalnya `nim_nama_crud_db.sql`.

## Catatan

- SQL dump yang dikumpulkan harus berasal dari database setelah semua misi CRUD selesai.
- Nama file screenshot bebas, tetapi harus mudah dipahami.
- Jika dosen/asisten meminta, sertakan juga file `.env` versi yang sudah disamarkan tanpa password asli.
