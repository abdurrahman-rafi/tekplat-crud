# Format Submission

## Yang wajib dikumpulkan

1. Screenshot bukti pembuatan database pada phpMyAdmin di aaPanel, dengan format nama `nim_nama_BuktiDB.jpg`
2. Screenshot halaman web yang sudah bisa diakses dari browser melalui domain/IP VM, dengan format nama `nim_nama_BuktiWeb.jpg`.
3. Screenshot bukti operasi Create, dengan format nama `nim_nama_BuktiCreateDina.jpg` dan `nim_nama_BuktiCreateRaka.jpg`.
4. Screenshot bukti operasi Read, dengan format nama `nim_nama_BuktiRead.jpg`.
5. Screenshot bukti operasi Update, dengan format nama `nim_nama_BuktiUpdate.jpg`.
6. Screenshot bukti operasi Delete, dengan format nama `nim_nama_BuktiDelete.jpg`.
7. File SQL dump hasil akhir database, dengan format nama `nim_nama_crud_db.sql`.
8. Semua bukti di-*compile* pada satu file zip, dengan format nama `nim_nama_TugasCRUD-Go.zip`

## Catatan

- SQL dump yang dikumpulkan harus berasal dari database setelah semua misi CRUD selesai.
- Jika dosen/asisten meminta, sertakan juga file `.env` versi yang sudah disamarkan tanpa password asli.
